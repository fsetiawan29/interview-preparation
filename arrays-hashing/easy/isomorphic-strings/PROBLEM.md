# Examples

**Example 1:**

Input: s = "egg", t = "add"
Output: true

**Example 2:**

Input: s = "foo", t = "bar"
Output: false

**Example 3:**

Input: s = "paper", t = "title"
Output: true

# Constraints

- `1 <= s.length <= 5 * 10^4`
- `t.length == s.length`
- `s` and `t` consist of any valid ASCII character.
