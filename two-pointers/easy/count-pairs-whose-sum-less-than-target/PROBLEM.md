# Examples

**Example 1:**

Input: nums = [-1,1,2,3,1], target = 2
Output: 3
Explanation: There are 3 pairs of indices that satisfy the condition: (0,1), (0,2), (0,4) since 0 + 1 < 2, 0 + 2 < 2, and 0 + 4 < 2.

**Example 2:**

Input: nums = [-6,2,5,-2,-7,-1,3], target = -2
Output: 10

# Constraints

- `1 <= nums.length <= 50`
- `-50 <= nums[i], target <= 50`
