# Examples

**Example 1:**

Input: s = "IceCreAm"
Output: "AceCreIm"
Explanation: The vowels in s are ['I', 'e', 'e', 'A']. On reversing the vowels, s becomes "AceCreIm".

**Example 2:**

Input: s = "leetcode"
Output: "leotcede"

# Constraints

- `1 <= s.length <= 3 * 10^5`
- `s` consists of printable ASCII characters.
