# Examples

**Example 1:**

Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
Explanation: The distinct triplets are [-1,0,1] and [-1,-1,2].

**Example 2:**

Input: nums = [0,1,1]
Output: []
Explanation: The only possible triplet does not sum up to 0.

**Example 3:**

Input: nums = [0,0,0]
Output: [[0,0,0]]

# Constraints

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`
