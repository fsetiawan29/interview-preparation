# Examples

**Example 1:**

Input: height = [1,8,6,2,5,4,8,3,7]
Output: 49
Explanation: The max area of water the container can contain is between index 1 and index 8 (height[1] = 8, height[8] = 7), giving an area of min(8, 7) * (8 - 1) = 49.

**Example 2:**

Input: height = [1,1]
Output: 1

# Constraints

- `n == height.length`
- `2 <= n <= 10^5`
- `0 <= height[i] <= 10^4`
